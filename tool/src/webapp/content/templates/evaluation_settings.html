Evaluation Dates:
Start Date: 
Users may begin submitting evaluation responses on this date
Due Date: 
Evaluation ends on this date, no more responses after this date
Stop Date: 
Users may still submit responses up until this date (defines a grace period)
View Date: 
Results are not viewable until this date
Evaluation results viewable by:
 Private (owner only - Note: Checking the private box overrides settings below
 Students in the courses this evaluation is assigned to Results viewable on "View Date"
 Instructors in the courses this evaluation is assigned to Results viewable on "View Date"
* Note that admins above in the hierarchy can view the results UNLESS the private box is checked
