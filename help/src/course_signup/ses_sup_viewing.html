Viewing Your Students Attending Modules

Supervisors are able to view a list of their students who are enrolled on courses throughtout the Division.

  1. To view module attendance, go to Module Signup on the left hand side navigation menu, then click the Module Administration link at the top.
  2. A list of students will appear and these can be sorted by status using the filters in the signups header.
  3. You have the option of changing a student's status in this screen by using the drop down next to each name.
