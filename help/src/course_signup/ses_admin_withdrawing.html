Withdrawing a Student from a Module

Module Administrators may find it necessary from time to time to withdraw a student during a later stage in the enrolment process.

  1. Click on Module Administration at the top of the Module Signup page.

  2. Select the Module from which the student wishes to withdraw from the drop-down list at the top of the page underneath Module Administration.

  3. Go to the student's entry under attendees (the second table down) and select WITHDRAWN from the drop-down box.

  4. This will change the student's status to withdrawn on the system.
